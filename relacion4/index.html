1. Introducción a Cookies y Sesiones

Aplicación PHP básica en 2 archivos:

  • login.php: una página html con un formulario de acceso que pide id de usuario y contraseña
  • proceso.php: crea una sesión, comprueba que el id de usuario y la contraseña son los que deben ser invocando a una función PHP con valores locales para ambos datos y establece una cookie con el nombre y una variable de sesión:
    setcookie("id_usuario", $nombre_usuario, time() + 30, "/"); $_SESSION["id_usuario"] = $_POST['nombre_usuario'];
A continuación, mediante dos echo, muestra el nombre a través de cookie y el nombre a través de variable de sesión

2. Variables de Sesión

Demuestre de forma práctica el uso de las variables de sesión y su mantenimiento. Para ello, se muestran dos variables diferentes: num1 y num2, que inicialmente, valdrán 0 ambas, y un campo select de formulario en el que podremos elegir cuatro opciones:

  • Incrementar a
  • Decrementar a
  • Incrementar b
  • Decrementar b
3. Juego de Adivinar un Número

El servidor "piensa" un número entre 1 y 100. El usuario introduce mediante un formulario, el número que cree que ha "pensado" el servidor. Si ha acertado, recibe le enhorabuena, si no, una pista: "te has pasado" o "te has quedado corto", ofrecerá un nuevo intento al usuario.

Tanto formulario como procesamiento en un mismo archivo en un mismo archivo. Método get para el paso de valores, y para mantener el número entre varias llamadas, el paso de campos de formulario tipo hidden.

4. Juego de Adivinar un Número v1

Segunda versión del programa anterior, pero estta vez, en vez de usar recursos de campos hidden, se usa el establecimiento de sesiones y de variables de sesiones.

5. Clase Restaurante

Programa PHP que define la clase Restaurante, con los siguientes atributos:

  • nombre
  • tipoCocina
  • ratings (es un array de valoraciones numéricas entre 1 y 5)
Define los métodos:
  • constructor (solo incluye como parámetros el nobre y el tipoCocina e inicializa ratings a vacío)
  • destructor
  • toString (permite visualizar la información de forma estructurada)
  • obtener el número de ratings
  • añadir un rating
  • añadir otros ratings (más de uno)
  • calcular rating medio
6. Clase Restaurante v1

Modificación del ejercicio anterior para que todos los atributos sean privados y solo se pueda acceder a ellos mediante getters y setter.
Bonus:
Creación de atributo static de clase: numeroRest, privado, que se incremente con cada nuevo objeto y se pueda acceder a él mediante un método público de clase totalRest()

13. Funciones de Cadena

Pide una cadena de texto por formulario de entrada y muestra en el interior de un alert de Bootstrap diferente, cada uno de los siguientes resultados:

  • La misma cadena de caracteres, del revés, indicando además si es palíndroma
  • La misma cadena de caracteres, con las palabras del revés
  • La misma cadena de caracteres toda en mayúsuclas y toda en minúsculas
  • Un recuento de caracteres (incluidos los espacios en blanco) y de las palabras que contiene
  • Como curiosidad, el resultado de aplicarle crypt, md5 y sha1
14. Funciones Anónimas

  • $circulo = function ($n) { ... } (calcula el área de un círculo)
  • $esfera = function ($n) { ... } (calcula el volumen de una esfera)

16. Funciones Callback Arrays

Programa sin entrada de datos, con un array numérico definido con la función range() y un rango de valores entre 1 y 100. Realiza las siguientes operaciones:

  • Aplica array_all para comprobar si todos los números son positivos
  • Aplica array_any para comprobar si hay algún múltiplo de 5
  • Aplica array_filter para extraer los que sean primos
  • Aplica array_find para dar la primera ocurrencia de número de dos cifras idénticas
  • Aplica array_map para obtener el cuadrado de cada valor
  • Aplica array_walk para sustituir cada vaor por su doble
Muestra cada resultado en un alert Bootstrap
